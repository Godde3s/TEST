# فروشگاه آنلاین (Online Store)

یک پروژه فروشگاه آنلاین مدرن و زیبا که با React و Vite ساخته شده است.

## ویژگی‌ها

- **صفحه اصلی** - نمایش محصولات ویژه و ویژگی‌های فروشگاه
- **صفحه محصولات** - نمایش تمام محصولات با قابلیت فیلتر بر اساس دسته‌بندی
- **صفحه جزئیات محصول** - نمایش اطلاعات کامل هر محصول
- **سبد خرید** - مدیریت آیتم‌های سبد خرید با امکان تغییر تعداد و حذف
- **صفحه تسویه خرید** - فرم ثبت اطلاعات ارسال و خلاصه سفارش

## تکنولوژی‌های استفاده شده

- **React 19** - کتابخانه UI
- **Vite** - ابزار ساخت و توسعه
- **React Router DOM** - مسیریابی
- **Lucide React** - آیکون‌ها
- **CSS3** - استایل‌دهی

## نصب و راه‌اندازی

```bash
# ورود به پوشه پروژه
cd frontend

# نصب وابستگی‌ها
npm install

# اجرای سرور توسعه
npm run dev

# ساخت نسخه تولید
npm run build

# پیش‌نمایش نسخه تولید
npm run preview
```

## ساختار پروژه

```
frontend/
├── src/
│   ├── components/          # کامپوننت‌های قابل استفاده مجدد
│   │   ├── Header.jsx       # هدر سایت
│   │   └── ProductCard.jsx  # کارت محصول
│   ├── context/             # Contextهای React
│   │   └── CartContext.jsx  # مدیریت سبد خرید
│   ├── pages/               # صفحات اصلی
│   │   ├── Home.jsx         # صفحه اصلی
│   │   ├── Products.jsx     # لیست محصولات
│   │   ├── ProductDetail.jsx # جزئیات محصول
│   │   ├── Cart.jsx         # سبد خرید
│   │   └── Checkout.jsx     # تسویه خرید
│   ├── App.jsx              # کامپوننت اصلی
│   └── main.jsx             # نقطه ورود
└── package.json
```
